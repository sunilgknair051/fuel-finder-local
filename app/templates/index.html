How privacy works

There are no accounts, cookies, analytics, advertising, tracking, persistent history, or browser storage. The raw postcode is converted to coordinates using the bundled local GeoNames-derived dataset and is not sent to Tankerkönig.

To retrieve prices, the server necessarily sends latitude, longitude, radius, and the operator's API key to Tankerkönig. The network connection also exposes the app host's IP address in the normal way. It would therefore be inaccurate to claim that absolutely no information leaves the device.
